Principais formatações

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo, temos outro texto marcado no final.

Texto pequeno

Estamos criando um texto pequeno com a tag <small>.

Texto deletado

Podemos cria um texto deletado, que deve ser lido mais não considerado.

Texto inserido

Podemos marcar um texto com inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins> (semântica).

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisa do tipo x 20 + 3. Usamos a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H2O. Usamos a tag <sub>.
